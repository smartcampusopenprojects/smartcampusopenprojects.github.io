No que consiste

As bibliotecas ajudam a educar a população em geral, além de fornecer espaços cívicos para interação. O número de usuários ativos de bibliotecas é uma medida do alcance e da eficácia das bibliotecas locais.

Alinhamento com a ISO 37122

Um usuário ativo de biblioteca pública deve se referir a um membro da biblioteca pública registrado a uma conta de usuário em uma biblioteca pública que utilize frequentemente serviços de biblioteca com pelo menos uma transação por mês, como acessar bancos de dados on-line/biblioteca para baixar artigos ou e-books que requerem privilégio de usuário de biblioteca ou assinatura de livros da biblioteca. Os usuários ativos de biblioteca contabilizados neste indicador devem incluir apenas os usuários que residem na cidade. A cidade deve contar quaisquer usuários ativos de biblioteca dentro dos limites da cidade. Usuários de fora dos limites administrativos serão excluídos.

A cidade deve incluir neste indicador o número de usuários ativos da biblioteca que são membros das bibliotecas públicas, ou que são conhecidos de forma mensurável para acessar os referidos serviços de biblioteca. Uma biblioteca pública consiste em qualquer biblioteca que tenha uma coleção organizada de materiais impressos ou de outra biblioteca, ou uma combinação destes, e que seja financiada no todo ou em parte com fundos públicos e tenha um cronograma estabelecido em que os serviços estejam disponíveis ao público.

Projeto Open Source

O projeto Smart Library foi desenvolvido e publicado no artigo Development of an IoT-enabled Smart Library Systemfor a University Campus por Yash Gupta Sungkur. O artigo descreve a implementação de um sistema de biblioteca inteligente em um campus universitário usando dispositivos IoT. O sistema faz uso de análises e aprendizado de máquina para analisar tendências e fazer previsões. O protótipo do sistema é apresentado no artigo.

O projeto trata-se de um sistema de check-out inteligentes equipadas com leitores RFID para facilitar o processo de empréstimo. Após retirar os itens desejados das prateleiras, o usuário tocará no material, que possui uma etiqueta RFID embutida e validará seu rosto por meio de reconhecimento facial para registrar a reserva no banco de dados da biblioteca. Além disso, haverá controles na cabine de check-out para permitir transações, somente quando o item tiver sido reservado pelo usuário e ele não tiver emprestado mais de dois itens. Isso reduzirá consideravelmente os erros humanos no processo. O processo de reserva começa dentro do aplicativo da web depois que o usuário faz login e decide reservar um material. O usuário busca o livro e posteriormente, o sistema verifica sua disponibilidade e verifica o limite de reservas do usuário. Caso o usuário seja elegível, mas o item não esteja disponível, o usuário pode pedir para ser notificado sobre a disponibilidade do livro. Caso contrário, o livro é reservado para o usuário específico. O usuário pode então escolher o livro na biblioteca
